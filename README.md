# wgpu_l3

A desktop UI framework built on wgpu and winit: an Elm-style value-semantics
runtime, a typed command system with claims and availability, native popup
windows with OS-composited materials, virtualized provided containers, and
record tables whose capabilities derive from std traits (`Display`,
`FromStr`, `Ord`) rather than framework vocabulary.

The governing design document is [docs/master_design.md](docs/master_design.md);
tracked future work lives in [docs/roadmap.md](docs/roadmap.md); audit and
campaign ledgers live in [docs/audits/](docs/audits/).
